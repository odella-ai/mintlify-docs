## Find Document

Finds a single document in a document store by matching metadata fields.
Add match criteria to create input ports — wire values from upstream nodes
to look up documents by case number, reference ID, client name, etc.

## Block metadata

| Field | Value |
| --- | --- |
| Block ID | `findDocument` |
| Display name | Find Document |
| Group | Documents |
